# Décisions techniques issues du benchmark

## Conservé comme cœur du projet
- **YOLO Objects** : retenu comme base de détection de personnes et lecture de scène.
- **YOLO Pose** : retenu comme extension utile pour enrichir l’analyse.

## Conservé mais à usage ciblé
- **Face Recognition** : conservé pour les démonstrations d’identification.
- **Face Tracking** : conservé comme solution simple ou de secours.
- **Hand Recognition** : conservé comme module complémentaire et expérimental.

## À utiliser avec prudence
- **Shape + Face** : utile en démonstration, mais plus coûteux.
- **Combined Recognition** : très intéressant pour illustrer la richesse du projet, mais à limiter dans un scénario continu.

## Position finale
Le benchmark n’invalide pas les modules riches, mais montre qu’ils doivent être utilisés avec discernement et non comme configuration permanente.
