# Résumé exécutif — Objectif obligatoire 1

L’objectif obligatoire 1 consistait à structurer une veille technologique et à produire un benchmark utile à la décision sur les briques logicielles du projet Over Sight. Le travail réalisé montre que le projet dispose déjà :
- d’un ensemble cohérent de modules de vision,
- d’outils de lancement et de démonstration,
- d’un système de benchmark avec rapport,
- d’une base suffisante pour hiérarchiser les choix techniques.

La veille permet de justifier les technologies explorées. Le benchmark permet de sortir d’un discours uniquement descriptif pour montrer ce qui fonctionne le mieux, ce qui coûte le plus cher et ce qui doit être priorisé dans la suite du projet.

La conclusion générale est claire : la détection de personnes constitue le socle principal, les modules de pose et de visage apportent une réelle valeur d’enrichissement, tandis que les modes combinés doivent être utilisés avec discernement en raison de leur coût.
