# CI et qualité — Objectif 2

## Objectif
Mettre en place une vérification automatique simple mais crédible.

## Outils
- Black
- Ruff
- pytest
- GitHub Actions

## Principe
La CI proposée ne cherche pas à exécuter toutes les démos.
Elle vérifie surtout :
- la cohérence de certains fichiers Python clés,
- quelques helpers testables,
- une partie de la structure projet.

## Bénéfices
- meilleure crédibilité du dépôt,
- prévention de certaines erreurs simples,
- base solide pour des évolutions futures.
