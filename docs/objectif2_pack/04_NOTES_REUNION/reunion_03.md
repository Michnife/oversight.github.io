# Réunion 3

## Sujet
Définition du périmètre de la CI.

## Résumé
Décision de ne pas lancer tous les modules lourds dans la CI.
La CI doit rester stable et viser les éléments les plus reproductibles : helpers, structure, scripts non interactifs.

## Conclusion
CI Python minimale retenue, orientée lint + tests d’infrastructure.
